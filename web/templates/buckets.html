No buckets yet. Create one above or use the S3 API.

Bucket names: 3-63 chars, lowercase letters, numbers, hyphens. Must start/end with letter or number. Names are globally unique within BucketCheap.

Manage buckets using the S3 API or AWS CLI:

# First, add to ~/.aws/config:

[profile bucketcheap]

s3 =

  addressing_style = path

# Create a bucket

$ aws s3 mb s3://my-bucket --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

# List buckets

$ aws s3 ls --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

# Upload a file

$ aws s3 cp myfile.mp4 s3://my-bucket/ --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

# Delete a bucket

$ aws s3 rb s3://my-bucket --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

⚠️ Path-style addressing required. Add addressing_style = path to your AWS profile or use forcePathStyle: true in SDKs.

Python (boto3)

s3 = boto3.client('s3',
    endpoint_url='https://s3.bucketcheap.com',
    aws_access_key_id='YOUR_KEY',
    aws_secret_access_key='YOUR_SECRET',
    config=Config(s3={'addressing_style': 'path'})
)

s3.create_bucket(Bucket='my-bucket')
s3.upload_file('video.mp4', 'my-bucket', 'video.mp4')

Node.js (AWS SDK v3)

import { S3Client, CreateBucketCommand } from '@aws-sdk/client-s3';

const s3 = new S3Client({
  endpoint: 'https://s3.bucketcheap.com',
  region: 'us-east-1',
  forcePathStyle: true,
  credentials: { accessKeyId: 'KEY', secretAccessKey: 'SECRET' },
});

await s3.send(new CreateBucketCommand({ Bucket: 'my-bucket' }));
