Buckets

Manage buckets using the S3 API or AWS CLI:

# Create a bucket

$ aws s3 mb s3://my-bucket --endpoint-url https://s3.bucketcheap.com

# List buckets

$ aws s3 ls --endpoint-url https://s3.bucketcheap.com

# Delete a bucket

$ aws s3 rb s3://my-bucket --endpoint-url https://s3.bucketcheap.com
