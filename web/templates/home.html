Get Started in 2 Minutes

# 1. Add this to ~/.aws/config

[profile bucketcheap]

s3 =

  addressing_style = path

# 2. Set your credentials

$ aws configure --profile bucketcheap

# 3. Create a bucket

$ aws s3 mb s3://my-bucket --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

# 4. Upload a file

$ aws s3 cp video.mp4 s3://my-bucket/ --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

# 5. List your buckets

$ aws s3 ls --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

Path-style addressing required (same as MinIO, R2, Backblaze). All S3 SDKs support it natively.
