Quick Start

Use these settings with any S3 client:

Endpoint: https://s3.bucketcheap.com

Region: us-east-1 (any value works)

Path Style: true (required)

Access Key: Generate in API Keys →

# AWS CLI — add to ~/.aws/config:

[profile bucketcheap]

s3 =

  addressing_style = path

# Then use:

$ aws s3 ls --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

$ aws s3 cp file.mp4 s3://my-bucket/ --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

⚠️ Path-style addressing is required (same as MinIO, R2, Backblaze). See API Keys for Python, Node.js, and rclone examples.
