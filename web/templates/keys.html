Quick Start

AWS CLI (~/.aws/config)

[profile bucketcheap]
s3 =
  addressing_style = path

# Then use:
aws s3 ls --endpoint-url https://s3.bucketcheap.com --profile bucketcheap

Python (boto3)

import boto3
from botocore.config import Config

s3 = boto3.client('s3',
    endpoint_url='https://s3.bucketcheap.com',
    aws_access_key_id='YOUR_ACCESS_KEY',
    aws_secret_access_key='YOUR_SECRET_KEY',
    config=Config(s3={'addressing_style': 'path'})
)

Node.js (AWS SDK v3)

import { S3Client } from '@aws-sdk/client-s3';

const s3 = new S3Client({
  endpoint: 'https://s3.bucketcheap.com',
  region: 'us-east-1',
  forcePathStyle: true,
  credentials: {
    accessKeyId: 'YOUR_ACCESS_KEY',
    secretAccessKey: 'YOUR_SECRET_KEY',
  },
});

rclone (~/.config/rclone/rclone.conf)

[bucketcheap]
type = s3
provider = Other
endpoint = https://s3.bucketcheap.com
access_key_id = YOUR_ACCESS_KEY
secret_access_key = YOUR_SECRET_KEY
force_path_style = true

⚠️ Path-style addressing required (forcePathStyle: true). Same as MinIO, Cloudflare R2, Backblaze B2.
